# A/B Test Line Chart

Interactive chart for A/B test conversion rate visualization.

## Tech

- React + TypeScript
- Recharts
- CSS Modules

## Features

- Conversion rate chart with multiple variations
- Hover tooltip with daily stats
- Day/Week toggle
- Variation selector
- Responsive (671-1300px)

### Bonus

- Zoom controls
- Line style (Line, Smooth, Area)
- Dark/Light theme
- PNG export

## Setup

```bash
npm install
npm run dev
```

Build: `npm run build`
